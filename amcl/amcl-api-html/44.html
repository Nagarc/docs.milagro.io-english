Decrypt data using AES GCM.
AES is run as a block cypher in the GCM mode of operation. The key size is 128 bits. This function will decrypt any
data length.
Parameters
K
128 bit secret key
IV
96 bit initialization vector
H
Additional authenticated data (AAD). This data is authenticated, but not encrypted.
C
Ciphertext.
Returns
P Decrypted data. It is the same length as the ciphertext.Plaintext
T 128 bit authentication tag.
