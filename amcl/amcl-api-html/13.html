Build Instructions
The default build is for 32 bit machines
1. mkdir Release
2. cd Release
3. cmake ..
4. make
5. make test
6. make doc
7. sudo make install
The build can be conﬁgured using by setting ﬂags on the command line i.e.
1. cmake -DWORD_LENGTH=64 ..

• sudo make uninstall
