Perform third pass on the server side of the 3-pass version of the M-Pin protocol.
If Time Permits are disabled, set d = 0, and UT and HTID are not needed and can be set to NULL. If Time Permits
are enabled, and PIN error detection is OFF, U and HID are not needed and can be set to NULL. If Time Permits
are enabled, and PIN error detection is ON, U, UT, HID and HTID are all required.

|
H(ID))
